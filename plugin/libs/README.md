# Godot Library Placeholder

Download the Godot library AAR file and place it here.

## Download Options:

### Option 1: From Godot Downloads
1. Go to https://godotengine.org/download
2. Download the Android export templates
3. Extract and find `godot-lib.*.aar`

### Option 2: From TuxFamily
Direct link for Godot 4.3:
https://downloads.tuxfamily.org/godotengine/4.3/godot-lib.4.3.stable.template_release.aar

### Option 3: From Your Godot Installation
1. Open Godot Editor
2. Export â†’ Android (create preset if needed)
3. Click "Manage Export Templates"
4. Find the templates folder
5. Look for `godot-lib.*.aar`

## File Naming
The file should be named like:
- `godot-lib.aar`
- or `godot-lib.4.3.stable.template_release.aar`

Any name matching `godot-lib*.aar` will work.
